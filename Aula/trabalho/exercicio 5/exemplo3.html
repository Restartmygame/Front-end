Desenvolva um script para validar a aprovação ou reprovação de um aluno em uma disciplina.
Para isso, crie uma página onde o usuário deve informar a nota 1 e a nota 2. Ao clicar em um botão, estes dados deverão
ser processados. Este script deverá calcular a somatória das duas notas (nota_final) e verificar:
Se nota_total 3,0 Então exibir mensagem de reprovado para o usuário
Se nota_total =7,0 Então exibir mensagem de aprovado para o aluno
Se nota_final =3,0 e 7,0 Então exibir mensagem de exame para o aluno.
